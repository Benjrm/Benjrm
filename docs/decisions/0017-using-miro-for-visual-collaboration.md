# Using Miro for Visual Collaboration

## Context and Problem Statement

The team needs a tool for visual collaborative work, specifically for conducting a competitive analysis and creating a story map with user stories.
The challenge is to find a tool that supports free-form visual layouts, real-time collaboration, and is easy to use without prior onboarding.

## Considered Options

* Miro
* Lucidchart
* draw.io

## Decision Outcome

Chosen option: "Miro", because it provides the most intuitive and flexible environment for the types of collaborative activities required in this project.

Miro offers a free tier that covers the needs of a small student team. Its infinite canvas, sticky notes, and built-in templates for story mapping and competitive analysis made it immediately usable without setup overhead.

Compared to Lucidchart, Miro is less rigid in its structure. Lucidchart is optimized for formal diagrams and flowcharts, whereas Miro's free-form canvas is better suited for the exploratory and collaborative nature of story mapping and competitive analysis workshops.

Compared to draw.io, Miro provides a more collaborative real-time experience and purpose-built templates for agile activities, reducing the effort needed to set up and run workshops.
