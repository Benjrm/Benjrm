# Using YouTrack for Project Management

## Context and Problem Statement

The team needs a tool for managing the project's agile workflow, including sprint planning, ticket management, and time tracking.

## Decision Outcome

YouTrack is used as the project management tool. It was prescribed by the university as the required platform for the project.

YouTrack covers all required project management needs: creating and managing tickets, organizing sprints, and tracking time spent on tasks. This allows the team to plan and document the agile development process in one place.
